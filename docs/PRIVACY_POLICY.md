# SkyPaste Privacy Policy

Effective date: 2026-03-20

SkyPaste is designed to keep clipboard history on your device.

## Data we store

SkyPaste stores clipboard history locally on the Mac where the app is installed. This may include:

- text copied to the clipboard
- image clipboard data or image previews
- file URL clipboard items
- item metadata such as timestamps, favorites, and local identifiers

## Data processing

SkyPaste processes clipboard data locally to provide:

- clipboard history browsing
- search and filtering
- favorites
- item deletion
- copy-back or paste-back actions

## Data sharing

SkyPaste does not upload clipboard history to a cloud service as part of the current version.

## Permissions

If you enable automatic paste-back, macOS may request Accessibility permission so the app can return content to the previously active app.

## Retention

Clipboard history remains on the device until you delete it in the app or uninstall the app.

## Contact

If you have questions about this policy, contact the developer through the repository or the app support page you publish for App Store submission.
